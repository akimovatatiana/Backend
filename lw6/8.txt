Выражение WHERE определяет предикат, то есть условия, которые должны быть выполнены для всех возвращаемых строк. 
Выражение HAVING применяется после группировки для определения предиката, фильтрующего группы по значениям агрегатных функций, 
то есть применяется после получения результата операции.
HAVING используется с SELECT и предложением GROUP BY, а выражение WHERE вместе с операциями SELECT, UPDATE, DELETE. 
WHERE не используется в запросах с агрегатными функциями.
